## ðŸš¦ Sample Query Dropdown Behavior

### Challenge:
Long queries (e.g. with binary fields like `picture`) can overflow or break layout in a `<select>` dropdown.

### Solution:

- âœ… Query dropdown options are truncated with `...` using JavaScript.
- âœ… Full query is shown in a `<textarea>` below the dropdown for editing and readability.
- âœ… `title` attribute is added so hovering the dropdown option shows full query.

### Future Improvement:
Want a smoother UX? Use a custom dropdown like `react-select` for wrapping, searching, tagging, and better scroll support.
